Google Web graph analysis

  • Tech Stack: PyTorch, NetworkX, Gephi

This project focuses on understanding the behavior and structure of the web using the Google Web Graph Data from the Google Programming Contest 2002. This analysis aimed to identify key structural characteristics of the web graph, including community detection and the presence of cliques.

Data Handling: Utilized NetworkX and Gephi to analyze the graph structure, initially focusing on a subset with 5,086 nodes and 43,589 edges. This involved preprocessing the data and performing degree-based sampling to simplify the dataset while maintaining its key properties.

Graph Analysis: Conducted an in-depth analysis of the graph's structure, revealing that it is densely connected with 56 communities and several large cliques. PageRank algorithms were compared using NetworkX, with Mean Squared Error as the comparison metric, including the baseline NetworkX PageRank, Hot Kernel PageRank, and Classic PageRank algorithms.

Link Prediction: Implemented and compared various link prediction algorithms, including Adamic-Adar, Jaccard Coefficient, and Preferential Attachment, using the NetworkX library to explore the potential for predicting new connections within the graph. Here, we also compared a Graph convolutional network's performance to traditional methods.
